## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (ursing the `pm install` command.
- Run the development web server using `node express_server.js`command

## Instrutions 

- You can use the short URLs putting `/u/<shortURL>` in your localhost.
